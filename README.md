# Maharashtra Forts Information Website

This project is a simple informational website developed using HTML.  
The website provides details about famous forts in Maharashtra along with images and historical information.

## Features
- Homepage with navigation to different forts
- Information pages for multiple forts
- Images of forts
- Simple and easy-to-use design
- Educational content about historical places

## Forts Included
- Raigad
- Rajgad
- Sinhagad
- Shivneri
- Sindhudurg
- Lohgad
- Torna
- Panhala
- Janjira
- Vishalgad
- Ramshej
- Harihargad
- Ajinkyatara
- Purandar

## Technologies Used
- HTML
- Basic Web Design
